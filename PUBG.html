配置要求及推荐

为了体验绝地求生的真正魅力，感受身临其境的真实生存，建议您使用推荐或更好配置进行游戏 （目前仅支持64位系统）

  • 最低配置
  • 普通配置
  • 推荐配置
  • 顶级配置

（最低能运行的配置，可能会卡顿 建议设置较低画质）

  • 系统 Windows7/Windows10 64位
  • 内存 6GB
  • CPU Intel i3-4340/AMD FX-6300
  • 显卡 GTX 660 HD 7850
  • 硬盘 7200转机械硬盘

（体验流畅画面，建议设置中等画质）

  • 系统 Windows7/Windows10 64位
  • 内存 8GB
  • CPU Intel i5-6400/AMD R3-1300X
  • 显卡 GTX 1050 / GTX 960 / GTX 1050TI / RX 570 /span>
  • 硬盘 7200转机械硬盘

（体验高清画质，建议设置较高画质）

  • 系统 Windows7/Windows10 64位
  • 内存 16GB
  • CPU Intel i5-7600K/AMD R5-1500X
  • 显卡 GTX 1060 （荐）/ GTX 970 / RX 580
  • 硬盘 7200转机械硬盘

（感受身临其境，建议设置超高画质）

  • 系统 Windows7/Windows10 64位
  • 内存 16GB及以上
  • CPU Intel i7-7700K/AMD R7-1800X
  • 显卡 GTX 1070 / GTX1070TI / GTX 1080 / GTX 1080TI / RX Vega64/ RX Vega56
  • 硬盘 SSD（固态硬盘）
